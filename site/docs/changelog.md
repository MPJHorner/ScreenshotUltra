# Changelog

The authoritative changelog lives at
[`CHANGELOG.md`](https://github.com/MPJHorner/ScreenshotUltra/blob/main/CHANGELOG.md)
in the repo. Release notes are also attached to each tagged
[GitHub release](https://github.com/MPJHorner/ScreenshotUltra/releases).

Versioning follows [SemVer](https://semver.org/). The site is regenerated
on every push to `main`, so this page matches the latest changelog
on the default branch.
